# Bem-vindo ao Venus

Venus eh uma biblioteca de componentes angular desenvolvida
pelo time de front-end do Edu Nai.

## Installation
Copie e cole o comando em seu terminal dentro da pasta do seu projeto:
```
npm install @kauelima21/venus
```

## Motivation

A lib foi desenvolvida a partir da necessidade de padronizar e documentar
componentes reutilizaveis presentes no Edu Nai S.A. Facilitando a codificao e a
utilizacao do Desing System do produto.
